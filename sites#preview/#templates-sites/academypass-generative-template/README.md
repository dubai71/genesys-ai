# AcademyPass Generative Template

Template estatico baseado no redesign do AcademyPass/Human Academy.

## Como abrir

Use o servidor SPA incluso para rotas React funcionarem:

```powershell
cd C:\AI\genesys\sites#preview\#templates-sites\academypass-generative-template
python serve_spa.py 5178
```

Depois abra:

```text
http://127.0.0.1:5178/
```

## Arquivos principais

- `index.html` carrega o app e o CSS de redesign.
- `assets/index-C5LpRRL3.css` e `assets/academypass-redesign.css` compoem o visual.
- `assets/*.js` contem o app React buildado e corrigido para rodar localmente.
- `hero/`, `marcas/`, `starter/`, `banner.jpg` e videos sao os assets visuais usados.
- `serve_spa.py` serve `index.html` como fallback para rotas SPA.

## Direcao visual

Creative AI system: base escura, midia generativa, acentos lime/ciano/coral, cards tecnicos e linguagem de plataforma criativa.
